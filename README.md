# 📱 Pokédex App

Aplicativo mobile de Pokédex desenvolvido com **React Native** e **Expo**, consumindo a [PokéAPI](https://pokeapi.co/) para buscar informações em tempo real sobre qualquer pokémon.

---

## 📸 Preview

> Busque qualquer pokémon e veja suas informações instantaneamente!

---

## ✨ Funcionalidades

- 🔍 Busca por nome do pokémon
- 🖼️ Exibição da imagem oficial
- ⚖️ Peso em quilogramas
- 🏷️ Tipos com cores dinâmicas (fogo, água, dragon, etc.)
- ⚔️ 5 movimentos principais

---

## 🚀 Como rodar o projeto

### Pré-requisitos
- [Node.js](https://nodejs.org/) instalado
- [Expo Go](https://expo.dev/client) instalado no celular

### Passo a passo

```bash
# Clone o repositório
git clone https://github.com/andrelmarques271-SD/Pokedex-App---React-Native

# Entre na pasta do projeto
cd Pokedex-App---React-Native

# Instale as dependências
npm install

# Rode o projeto
npx expo start
```

Escaneie o QR Code com o **Expo Go** e o app abrirá no seu celular!

## 📚 O que aprendi desenvolvendo esse projeto

- Gerenciamento de estado com `useState`
- Requisições assíncronas com `async/await` e `fetch`
- Tratamento de erros com `try/catch`
- Renderização de listas com `.map()` e `.slice()`
- Estilização com `StyleSheet` do React Native
- Estilos dinâmicos baseados em dados da API
